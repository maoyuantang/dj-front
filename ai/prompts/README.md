# 任务 Prompt 模板

- 用途：帮助人把单次任务的目标、输入、约束、步骤和验收说完整。
- 边界：允许可复用的 `{{placeholder}}` 模板；不保存长期项目事实、真实密钥或某次任务结果。
- 读取时机：任务描述不完整时选择一个最接近的模板，并与相关 workflow 一起使用。
- 更新与责任：任务类型、必填输入或输出要求变化时，由经常执行该任务的人更新。
- 权威来源：行为约束仍来自 [`../rules/`](../rules/README.md)，流程来自 [`../workflows/`](../workflows/README.md)。
- 项目示例：新增页面使用 `create-page.md`，Bug 诊断使用 `fix-bug.md`；不要把一次页面名称写回模板。

简单任务可直接引用模板内容，不必为了形式新建记录文件。
